
  # Sanskrit Language Learning Website

  This is a code bundle for Sanskrit Language Learning Website. The original project is available at https://www.figma.com/design/QbNEj62EcOiUodWXbf4ZsQ/Sanskrit-Language-Learning-Website.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Cloudflare D1 Lexicon

  D1 schema, ingestion scripts, and operational notes live in `../docs/d1-wordlist.md`. Run `npm run d1:setup` and `npm run d1:ingest -- --file <path> --database vaan_sanskrit_lexicon` when you need to update the Sanskrit wordlist.
